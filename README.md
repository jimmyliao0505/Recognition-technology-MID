🚗 動機與目的

隨著自駕與駕駛輔助技術的發展，車道辨識已成為先進駕駛輔助系統（ADAS）中不可或缺的一環。
本專案的目的是利用 OpenCV 實作一個能即時偵測車道線的系統，協助車輛判斷行駛區域與中心位置，
在不依賴深度學習的情況下完成車道辨識任務，適合早期開發階段或資源有限的應用。

🧠 方法：程式

使用 OpenCV 讀取影片逐幀處理。

對影像進行 灰階化與高斯模糊，減少雜訊干擾。

使用 Canny 邊緣偵測 找出可能的車道線邊緣。

限制偵測區域（ROI）在畫面下半部，提高效率並排除無關區域。

透過 Hough 轉換 找出左右車道線位置。

計算車道中心並與車輛中心比較，標示偏移情況。

將結果以綠色半透明區域疊加在畫面上，強化辨識效果。

⚙️ 優化程式地方

ROI 區域限制：只偵測畫面下半部，減少不必要運算。

左右線條補強：當虛線中斷時，強制從畫面下方延伸偵測線，避免車道消失。

透明疊加顯示：使用加權合成提升可視性，同時保留原始影像細節。

影片輸出功能：可將即時辨識結果輸出為影片，便於測試與分析。
